# 技术栈

- React
- Redux
- React-Router
- Ant Design
- Ant Design Charts
- styled-components
- axios
- ESLint

# 运行

1. 先运行[后端](https://github.com/cpp233/node-backend)

2. 进入前端目录运行

```shell
npm i
npm start
```
